# Day 3 – Assignment 3 (Next.js App Router pages & Dynamic routing )

## Overview
This project is a simple multi-page application built using **Next.js App Router**.  
It demonstrates folder-based routing, shared layouts, and client-side navigation with `Link`.

## Pages Implemented
- **Home (`/`)** – Main landing page  
- **About (`/about`)** – Information about the project  
- **Contact (`/contact`)** – Contact details or form  

## Features
- **Folder-based routing** for clean URL structure  
- **Shared layout (`layout.js`)** for Navbar and Footer  
- **Navigation** using `Link` from `next/link`  

## Additional Implementation
- **Dynamic routing** with `/products/[slug]`  
- Product listing and individual product detail pages  
- Graceful **404 handling** with `notFound()`  

## Technologies Used
- Next.js (App Router)  
- React  
- Tailwind CSS  
- JavaScript (ES6+)  

## Outcome
The assignment satisfies all core requirements and includes additional dynamic routing to demonstrate understanding of Next.js routing concepts.
